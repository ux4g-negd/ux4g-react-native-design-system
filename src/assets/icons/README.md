# Custom Icons

Place your custom SVG or PNG icons here.

## Usage in Flutter

```dart
import 'package:flutter_svg/flutter_svg.dart';

// SVG icon
SvgPicture.asset('assets/icons/your_icon.svg', width: 24, height: 24)

// PNG icon
Image.asset('assets/icons/your_icon.png', width: 24, height: 24)
```

For custom icon fonts, follow Flutter's icon font documentation:
https://flutter.dev/docs/development/ui/widgets/icon
